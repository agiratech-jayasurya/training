What is CSS ?

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls how elements on a webpage should be displayed ‚Äî including layout, colors, fonts, spacing, animations, and responsiveness.

üß± 1. CSS Syntax, Selectors, Specificity

Syntax: A CSS rule is made up of a selector and a declaration block.

Selectors: Target specific HTML elements.
Specificity: Spcifically target the element by declaring the ID and Class name to specific element. 

  
    // By the selector, directly access the tag.
    h1 {
      color: blue;
      font-size: 24px;
    }

    p {
      margin: 10px;
    }

    // Declare the ID name, accessing through it.
    #main-title {
      color: red;
    }

    // Declare the class name and access through it.
    .highlight {
      background: yellow;
    }

üß© 2. Types of CSS (inline, internal, external)

CSS can be applied in three ways: inline, internal, and external.

Inline : Inline CSS is applied directly to an individual HTML element using the style attribute. It only affects that specific element.

Internal : Internal CSS is written within a style tag inside the header tag section of an HTML document. It applies to the entire page.

External : External CSS is written in a separate .css file and linked to an HTML file using the link tag. It can be reused across multiple pages.

üìç 6. Positioning (static, relative, absolute, fixed, sticky)

Controls how elements are positioned in the document flow:

static : default

relative : relative to its normal position

absolute : positioned relative to nearest positioned ancestor

fixed : relative to viewport

sticky : stick the element specific position
